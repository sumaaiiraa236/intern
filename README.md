Intern Dashboard
A dummy login and dashboard website for interns.
Features

- Dummy Login / Signup (no real auth)
- Dashboard with:
  - Intern name
  - Referral code (e.g., sumairaa2025)
  - Total donations raised
  - Rewards/unlockables (static display)
How to Run
Frontend:
- Go to `frontend/`
- Open `index.html` in your browser
Backend:

cd backend
npm install
npm start
